Tools we like

  • Scapy
    A fantastically capable tool for crafting packets by hand.

  • GitBucket
    A Github clone written in Scala. Perfect for a private Git repository on a low cost DigitalOcean or AWS server. The primary developer, Naoki Takezoe, places a strong emphasis on ease of installation which I absolutely love.

  • Mongoose
    A powerful asyncronouns, non-blocking web server designed for use in embedded systems. Supports CGI, SSL, Digest auth, Websocket, WEbDAV, Resumed download, URL rewrite, file blacklist, IP-based ACL, and available as a single file.

  • darkhttpd
    Also available as a single file - darkhttpd is has a simpler feature set compared to Mongoose, but very useful "when you need a web server in a hurry.."

Dns Resolver

You can use this free (rfc-compliant rfc1033, rfc1034, rfc1035) tool to perform dns queries using either our name servers, or those of your choosing. It supports common query types such as A, NS, CNAME, SOA, PTR, MX, TXT, RP, AXFR, AAAA, ANY, and more in addition to full support for IPv6.

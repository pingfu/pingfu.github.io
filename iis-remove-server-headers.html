Summary

A .Net module for IIS which removes the following unwanted HTTP headers from the server response.

  • Server
  • X-Powered-By
  • X-AspNet-Version
  • X-AspNetMvc-Version

Download

Either compile from source, or follow the links on GitHub to download pre-compiled binaries.

Installation to the GAC

Deploy the library to the Global Assembly Cache (GAC) using gacutil.exe. Once the library is installed to the GAC, other copies can be removed from the server.

gacutil.exe /i Pingfu.RemoveServerHeaderModule.dll

Configure IIS to use the module

  1. Open the Internet Information Services (IIS) Manager snap-in.
  2. Open the Modules administration feature at the server level.
  3. Select Add Managed Module...
    1. For Name enter a descriptive name, for example "Pingfu.RemoveServerHeaderModule"
    2. For Type select the module from the list.
